## 源码目录速览

本书分析的源码位于 `hermes-agent/` 仓库根目录，以下是核心模块结构：

```
hermes-agent/
├── run_agent.py          # AIAgent 类 — 核心会话循环 (10.6K 行)
├── model_tools.py        # 工具编排层 — 发现 + 分发
├── toolsets.py            # 工具集定义 — 40+ 工具的分组与组合
├── cli.py                 # HermesCLI 类 — 交互式 CLI 编排器 (10K 行)
├── hermes_state.py        # SessionDB — SQLite + FTS5 状态存储
├── agent/                 # Agent 内部模块
│   ├── prompt_builder.py      # System Prompt 组装 (含注入检测)
│   ├── context_compressor.py  # 上下文压缩引擎
│   ├── context_engine.py      # ContextEngine 抽象基类
│   ├── memory_manager.py      # 记忆管理器 (builtin + 外部插件)
│   ├── credential_pool.py     # 多凭据池 + 自动 failover
│   ├── anthropic_adapter.py   # Anthropic Messages API 适配器
│   ├── smart_model_routing.py # cheap/strong 模型路由
│   └── ...
├── tools/                 # 工具实现 (每工具一文件)
│   ├── registry.py            # 中央工具注册表
│   ├── terminal_tool.py       # 终端编排 (6 种后端)
│   ├── environments/          # 终端后端 (local/docker/ssh/modal/daytona/singularity)
│   ├── browser_tool.py        # 浏览器自动化
│   ├── mcp_tool.py            # MCP 客户端
│   ├── skills_tool.py         # Skills 系统
│   ├── delegate_tool.py       # 子 Agent 委派
│   └── ...
├── gateway/               # 消息平台网关
│   ├── run.py                 # GatewayRunner (9K 行)
│   ├── platforms/             # 15 个平台适配器
│   └── ...
├── hermes_cli/            # CLI 子命令系统
├── cron/                  # 定时任务调度器
├── plugins/               # 可插拔插件
├── skills/                # 内置 Skills
└── acp_adapter/           # ACP 服务器 (IDE 集成)
```

## 阅读约定

本书基于 **Python 源码**分析，使用真实的文件名、函数名、类名。标注格式如下：

> `AIAgent.run_conversation()` in `run_agent.py:7544` — 核心会话循环

- **前者**是源码中的**真实函数名**
- **后者**是**文件路径:行号**（基于 v0.8.0 版本）

每章末尾的**速查表**汇总了该章涉及的所有关键文件和函数。

## 阅读建议

- **快速了解**：先读第 3 章（十分钟心智模型），建立全局地图
- **深入核心**：接着读第 5 章（主循环解剖），理解 Agent 的心跳
- **按部分阅读**：每部分相对独立，可根据兴趣选择
- **源码对照**：每章都附有源码锚点，建议 clone 仓库后边读边对照
